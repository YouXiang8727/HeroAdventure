# 物品與商店設計手冊 (Item & Shop Design)

本文件記錄遊戲中所有可購買物品的種類、數值縮放邏輯以及商店的刷新機制。

## 1. 商店刷新機制 (Shop Mechanics)
*   **刷新規則**：每次進入商店，貨架上的商品都是隨機生成的。
*   **購買限制**：
    *   **屬性碎片**：**【限購一次】**。一旦購買，該商品會從當次商店貨架售罄。
    *   **藥水與裝備**：**【可重複購買】**。只要背包空間足夠且金幣充足，可以重複購買。

## 2. 消耗品 (Consumables)

### 生命藥水 (Health Potion) - 【可重複購買】
*   **基礎效果**：恢復生命值。
*   **數值縮放**：基礎值 (40 + 等級*8)。
*   **稀有度加成**：
    *   **普通 (Common)**：1.0x 效果 / 1.0x 價格
    *   **稀有 (Rare)**：3.0x 效果 / 2.2x 價格
    *   **史詩 (Epic)**：8.0x 效果 / 5.0x 價格
    *   **傳說 (Legendary)**：20.0x 效果 / 10.0x 價格

### 屬性碎片 (Stat Shard) - 【限購一次】
使用後永久提升英雄屬性。出現機率分配：攻擊 (40%) / 生命 (40%) / 爆擊 (20%)。

| 種類 | 稀有 (Rare) | 史詩 (Epic) | 傳說 (Legendary) |
| :--- | :--- | :--- | :--- |
| **攻擊碎片** | +4 ATK | +10 ATK | +24 ATK |
| **生命碎片** | +20 HP | +50 HP | +120 HP |
| **爆擊碎片** | +2% Crit | +5% Crit | +12% Crit |

## 3. 裝備系統 (Equipment) - 【可重複購買】
裝備分為**武器**與**防具**，數值受等級與稀有度影響。購買後進入背包，需在背包點擊使用才能裝備。

### 稀有度乘數 (Rarity Multipliers)
*   普通 (Common): 1.0x
*   稀有 (Rare): 2.0x
*   史詩 (Epic): 4.0x
*   傳說 (Legendary): 8.0x

### 裝備種類
*   **武器 (Weapons)**：
    *   **長劍 (Sword)**：平衡型攻擊力。
    *   **戰斧 (Axe)**：高攻擊力 (1.2x 價格)。
    *   **法杖 (Staff)**：較低攻擊力 (0.9x 價格)。
*   **防具 (Armor)**：
    *   **皮甲 (Light Armor)**：基礎生命值加成。
    *   **板甲 (Heavy Armor)**：高生命值加成 (1.5x 數值 / 1.3x 價格)。
