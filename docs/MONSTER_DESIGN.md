# 怪物設計手冊 (Monster Design)

本文件記錄所有怪物的種類、屬性成長公式以及特殊能力。

## 屬性成長公式
怪物的數值會隨著關卡等級 (Stage Level) 提升：
*   **難度乘數 (difficultyMultiplier)**：
    *   等級 1~30：`1.0 + (level - 1) * 0.12`
    *   等級 30 以上：`4.48 + (level - 30) * 0.40`
*   **隨機波動**：最終數值會有 ±5% 的隨機震盪。

## 怪物通用能力 (Abilities)
*   **吸血 (LifeSteal)**：攻擊時恢復傷害值一定比例的生命。
*   **反傷 (Thorns)**：受到攻擊時，反彈傷害值一定比例的數值給英雄。
*   **再生 (Regen)**：每回合（玩家行動後）恢復最大生命值一定比例的生命。
*   **狂暴 (Berserk)**：生命值低於門檻時，攻擊力提升（通常為 1.4~1.8 倍）。

## 特殊怪物：變形蟲 (Ditto)
變形蟲是極為特殊的怪物，出現機率固定為 **3%** (不受關卡等級影響)。

### 階級分配與數值複製
變形蟲會複製英雄「當下」的總數值（包含裝備與被動加成）：

| 階級 | 出現權重 | 複製比例 (HP/ATK) | 額外能力 |
| :--- | :--- | :--- | :--- |
| 初階變形蟲 | 48% | 70% | 無 |
| 高階變形蟲 | 37% | 100% | 再生 (5%) |
| 菁英變形蟲 | 15% | 120% | 再生 (8%), 反傷 (10%) |
